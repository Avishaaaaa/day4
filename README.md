Topics Covered
1. Objects

Objects are used to store multiple related values in the form of key-value pairs.

Used to group related data together
Values are accessed using dot notation
Properties can be updated anytime
Can also contain nested objects

2. Arrays

Arrays are used to store multiple values in a single variable.

Indexing starts from 0
Can store different types of data

Common methods:
push() → adds element at end
pop() → removes last element
shift() → removes first element
unshift() → adds element at beginning

3. Functions

Functions are reusable blocks of code used to perform specific tasks.

Types of Functions:
Function Declaration
Function Expression
Arrow Function

Concepts:
Parameters → input values passed to function
Return → output given back from function

Used for:
Performing calculations (add, multiply)
Checking conditions (even/odd)
Reusable logic in programs

4. Loops

Loops are used to execute a block of code multiple times.

Types of Loops:
for loop → used when number of iterations is known
while loop → runs while condition is true
do...while loop → runs at least once

Control Statements:
break → stops the loop completely
continue → skips the current iteration
